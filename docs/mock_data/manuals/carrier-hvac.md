# CARRIER COMMERCIAL HVAC MODEL-50TJ MANUAL

This manual provides troubleshooting protocols for the Carrier Model-50TJ Rooftop Air Handling Unit.

---

## 1. Temperature Control Issues

### Diagnostic: Blowing Warm Air (Cooling Mode)
If the unit is blowing warm air while set to cooling mode, perform the following troubleshooting steps:
1.  **Check Thermostat Settings:** Ensure the thermostat is set to "Cool" and the temperature setpoint is below current ambient air temperature.
2.  **Verify Compressor Status:** Check if the compressor is cycling. If the compressor does not cycle:
    *   **Error Code 1 flashes:** Indicates a High-Pressure Lockout. Reset the primary circuit breaker.
    *   **Error Code 2 flashes:** Indicates a Low-Pressure Lockout (potential refrigerant leak). Contact a certified technician.
3.  **Check Condenser Fan:** Ensure the condenser fan is spinning and clear of debris.
4.  **Inspect Evaporator Air Filter:** A clogged filter restricts airflow, causing the evaporator coils to freeze. Replace filters if dust accumulation exceeds 2mm.

---

## 2. Maintenance and Part Replacements
*   **Filter Size:** Carrier Model-50TJ requires standard 20x20x2 pleated MERV-8 air filters.
*   **Replacement Cycle:** Inspect every 90 days; replace at least every 180 days.
