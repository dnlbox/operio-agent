# Schindler Elevator Manual (Model-9300)

## Section 1 - Drive Motor Overtemperature

### Troubleshooting Code F-01
If the elevator system encounters Fault Code F-01, it indicates that the primary drive motor has exceeded safety temperature limits.
1. **Check Ventilation Fans:** Verify that the machine room ventilation fan is running and that ambient room temperature is below 35°C.
2. **Reset Overload Relay:** Locate the main cabinet control panel and inspect the thermal overload relay. Reset the switch manually once the motor has cooled.
3. **Inspect Brake Gap:** Ensure the mechanical brake calipers are not dragging against the flywheel.
