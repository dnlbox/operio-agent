# OTIS ESCALATOR MODEL-NPE SERVICE MANUAL

This manual provides service and diagnostic protocols for the Otis Model-NPE Heavy Duty Escalator.

---

## 1. Safety Interlocks and Error Codes

### Diagnostic: Escalator Stop and Fault Codes
If the escalator halts operations, check the primary diagnostic display located inside the upper truss access panel.

*   **Error Code E-01 (Broken Drive Chain):** Drive chain safety switch tripped. Escalator is locked. Do not attempt restart. Verify chain integrity.
*   **Error Code E-04 (Comb Plate Impact):** Comb plate safety switch tripped. Indicates an object (e.g., shoe, luggage strap) is caught between the steps and the comb plate.
    *   **Troubleshooting:**
        1. Turn off main power key-switch.
        2. Inspect comb teeth for obstruction; clear any debris.
        3. Verify comb plate alignment sensor clearance (must be 3.5mm).
        4. Reset switch and restart using the key-switch.
*   **Error Code E-09 (Skirt Safety):** Object caught between step and side skirt panel.
