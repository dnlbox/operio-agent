The Core Vision

Commercial retail properties are managed using fragmented, manual processes. When a tenant encounters an issue (e.g., HVAC failure, pipe leak), dispatchers spend hours manually auditing negotiated lease agreements to determine liability thresholds, checking vendor compliance, and drafting work orders.

Operio Agent is a cognitive orchestration layer built with Gemini and the Model Context Protocol (MCP). It automates the dispatcher reasoning loop, performing secure lease audits, on-site diagnostics, technician matching, and Human-in-the-Loop escalation.

The Polyglot Architecture

Our design showcases a high-performing polyglot architecture: a **Python agent brain** running Gemini and OpenTelemetry, managing **TypeScript MCP servers** running as stdio subprocesses for database and search operations.

  • Gemini (google-genai SDK)
  • Arize Phoenix (OTel Tracing)
  • Model Context Protocol (MCP)
  • FastAPI & Python 3.13
  • MongoDB (MCP Server)
  • Elasticsearch (MCP RAG Server)
  • TypeScript & Node.js

Every step of the agent's reasoning loop—from lease searches to MongoDB writes—is instrumented with OpenTelemetry and traced in Arize Phoenix for complete operational auditing.

Validated Hackathon Scenarios

1. Secure Tenant Lease Audit & Chargeback (Routine HVAC)

A tenant reports an AC failure. The agent executes a secure, tenant-isolated Elasticsearch RAG search. It detects Section 9.1 (Tenant responsible up to $1,000). The agent estimates cost ($850) and matches HVAC technician Sarah Connor. Because it is a tenant chargeback, it auto-dispatches, bypassing landlord approval gates.

2. Human-in-the-Loop (HITL) Gate (Routine Common Area)

Common area repair is requested, estimated at $250. Because the Landlord is responsible and the cost exceeds the $150 autonomous limit, the agent creates a ticket with "Pending Approval" status. The manager receives a notification and authorizes the dispatch, generating Yardi/ServiceChannel integration payloads.

3. Weather-Triggered Emergency Bypass (Burst Pipe)

During an extreme winter cold alert (-22°C), a tenant reports a burst pipe. The agent detects the critical temperature and incident severity, classifies it as an "Emergency," and triggers the Emergency Bypass. It auto-dispatches a plumber immediately to prevent structural damage, overriding standard landlord cost thresholds.

Developer Quick Start

Run the entire SRE command center locally using our environment setup:

# 1. Spin up MongoDB, Elastic, and Arize Phoenix docker compose up -d # 2. Seed database indexes and mock data npm run seed # 3. Launch the Python FastAPI backend & MCP client npm run dev

The PoC command center will be available at http://localhost:3001.
